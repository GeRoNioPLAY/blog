# Проект Блог

Проект для обучения базовым вещам FastAPI, SQLAlchemy, Alembic, Pydantic и т.д.

## Версия v0.1.2

- В версии v0.1.1 добавил pre-commit и task-runner (taskipy);
- В версии v0.1.2 поправил структура папок.

Сделал всю базовую логику с следующими функциями:

- create_post - создаёт пост с заголовком и текстом;
- get_posts - возвращает все посты;
- get_post_by_id - возвращает пост сразу с комментариями;
- create_comment - создаёт комментарий к указанному посту.
